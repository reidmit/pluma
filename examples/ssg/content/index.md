# Pluma

**Pluma** is a small, statically-typed functional language with
Hindley–Milner type inference, algebraic data types, and traits. This
very page was generated by a static site generator *written in Pluma*.

## Why another language?

Because building one teaches you what a language is *for*. Pluma leans
on a few ideas:

- everything is an expression
- pattern matching over `enum`s instead of `null`
- `result` and `option` instead of exceptions
- a small, composable standard library

> A language is a tool for thought. Keep it sharp, keep it small.
